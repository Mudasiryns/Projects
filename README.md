 PROJECT 1: Brain Tumor Classification using CNN

 This project classifies brain MRI images into:

Tumor
No Tumor

It helps in early diagnosis support using medical image classification.
Model Used
ResNet18
Framework Used
PyTorch
Dataset
Brain MRI Images for Tumor Detection
Task Type
Binary classification
Classes

Tumor
Non-Tumor

Important Steps

MRI images loaded
Images resized to 224×224
Normalization applied
Train/test split
ResNet18 transfer learning used
Model trained and evaluated
Confusion matrix and predictions shown
